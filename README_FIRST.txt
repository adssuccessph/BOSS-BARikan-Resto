BARIKAN EVENT BOOKING + ADMIN v3.2 — CATERING MENU BUILDER
===========================================================

OPEN THE DESIGN FIRST
---------------------
1. Extract this ZIP.
2. Double-click index.html for the CUSTOMER WEBSITE preview.
3. In the inquiry form, select Event Catering.
4. The Catering Menu section will automatically appear.
5. Double-click admin.html for the ADMIN DASHBOARD preview.
6. Open the Catering Menu tab to preview category limits and menu item management.

WHAT'S NEW IN v3.2
------------------
- Catering menu appears only when Event Catering is selected.
- No checkboxes: customer taps menu cards.
- Admin can add/edit/delete menu categories.
- Admin can set MAXIMUM MENU CHOICES per category.
  Example:
  Main Dish = max 2
  Noodles / Pasta = max 1
  Dessert = max 1
- Admin can add/edit/delete menu items.
- Each menu item can have a name, optional add-on price text, description, photo, active status and sort order.
- Customer cannot exceed the admin-set category maximum.
- Server also validates the maximum before saving the booking.
- Selected catering menu is saved in the Bookings sheet and shown in Admin Bookings.

LOCAL PREVIEW
-------------
The ZIP contains SAMPLE menu items only so you can see the customer and admin UI immediately.
These sample menu items are not automatically added to the live Google Sheet.
Add the real catering menu from Admin after deploying Code.gs.
